# RndmPswrdGnr

**Автор:** Шатилов Владимир

## Описание
Графическое приложение для генерации случайных паролей с настраиваемыми параметрами. Позволяет выбрать длину (от 1 до 128 символов) и типы используемых символов (цифры, буквы латиницы, спецсимволы). Все сгенерированные пароли сохраняются в историю (JSON-файл) и загружаются при следующем запуске. Реализованы очистка и удаление отдельных записей истории. Приложение снабжено валидацией ввода и информативными сообщениями об ошибках.

## Запуск
1. Клонируйте репозиторий
2. Запустите приложение (Python 3.8+): python main.py

Никаких дополнительных библиотек не требуется (tkinter входит в стандартную поставку Python).

## Использование
1. Выберите длину пароля с помощью ползунка.
2. Отметьте чекбоксы нужных категорий символов.
3. Нажмите «Сгенерировать пароль» – пароль появится в таблице истории.
4. История автоматически сохраняется в файл `history.json`.
5. Кнопка «Очистить историю» удаляет все записи (после подтверждения), «Удалить выбранное» – удаляет выделенные строки.

## Примеры
- Генерация пароля из 20 символов, только буквы и цифры.
- Пароль длины 8 со всеми типами.
- Попытка генерации без выбора символов вызывает ошибку.
- Попытка установить длину вне диапазона невозможна (ползунок ограничен).

## Структура проекта
- `main.py` – точка входа
- `history.json` – файл истории (создаётся автоматически)
